平假名

官方英文版更新于 2022-02-19 · 查看官方原文

平假名是日语的基本表音文字。它表示日语中的每一种声音。因此，从理论上说，你可以完全用平假名书写任何内容。然而，由于日语书写时不使用空格，这会产生几乎无法辨认的文本。

下面是一张平假名表，其中列出了发音相近的英语辅音—元音读音。阅读顺序是从上到下、从右到左，大多数日语书籍也是这样书写的。在日语中，按照正确的笔顺和方向书写笔画很重要，对汉字尤其如此。由于手写字符与印刷字符看起来略有不同（就像 ‘a’ 在印刷体中看起来完全不同一样），你需要使用一种以手写风格字体演示字符写法的资源（链接见下文）。我还必须强调正确学习每个音如何发音的重要性。由于日语中的每个词都由这些音组成，学错一个字符的发音会严重破坏你的整个发音基础。

掌握或教授平假名并不太难，因此网上已经有各种各样的网站和免费程序。我还建议你把自己的发音录下来并进行比较，以确保发音正确。

手写练习平假名时，需要牢记的重点是：笔顺和笔画方向非常重要。好了，我不仅加了下划线和斜体，还用了粗体，甚至又强调了一遍。相信我，当你看到别人匆忙写下、简直像鬼画符一样的笔记时，最终就会明白其中的原因。唯一能帮到你的是，每个人都按相同的顺序书写，因此字符的“走势”相当一致。我强烈建议你从一开始学习平假名时就密切注意笔顺，以免养成坏习惯。尽管网上有许多旨在帮助你学习平假名的工具，但学习书写的最佳方式还是传统方法：一张纸和一支钢笔或铅笔。下面是便于进行平假名书写练习的 PDF。

※ 顺带一提，直到近代，一首名为「いろは」的日本古诗还经常被用作平假名排序的基础。这首诗包含除「ん」以外的每一个平假名字符，因为「ん」在这首诗写成时可能还不存在。你可以在这篇维基百科文章中亲自查看这首诗。正如文章所说，这种顺序有时仍用于列表排序，因此你或许可以花些时间了解一下。

注意事项 

  1. 除「し」、「ち」、「つ」和「ん」之外，把顶行的辅音与元音相配，就能大致了解各个字符的发音。例如，「き」读作 / ki /，「ゆ」读作 / yu /，以此类推。
  2. 如你所见，并非所有发音都符合我们的辅音体系。正如表中所示，「ち」读作 “chi”，「つ」读作 “tsu”。
  3. 日语中的 / r / 或 / l / 音与英语中的任何发音都很不一样。发音时更需要卷一下舌，并用舌头敲击上颚，迅速将音截断。请特别留意这一整列。
  4. 请仔细注意 / tsu / 和 / su / 之间的区别。
  5. 「ん」是一个特殊字符，因为它很少单独使用，也没有元音。它附在另一个字符之后，以添加 / n / 音。例如，「かん」会读成 ‘kan’，而不是 ‘ka’；「まん」会读成 ‘man’，而不是 ‘ma’；以此类推。
  6. 你必须学习正确的笔顺和笔画方向！请使用以下 pdf 练习纸。

浊音

记住所有平假名字符后，仍有一些额外的发音需要学习。还有五组辅音，要在字符上添加两条类似双引号的小短线（称为 dakuten（濁点）），或者一个小圆圈（称为 handakuten（半濁点））来书写。这实际上会产生辅音的“浑浊”版本，或者说不那么短促的版本（专业上称为浊辅音或「濁り」，字面意思就是变得浑浊）。

  • 请注意，「ぢ」的发音实际上与「じ」相同，两者都读作 / ji /，而「づ」读作 / dzu /。

小写的「や」、「ゆ」和「よ」

还可以把小写的「や」、「ゆ」或「よ」附在各组辅音的 / i / 元音字符之后，将辅音与 / ya / yu / yo / 音组合起来。

平假名中小写 や、ゆ、和 よ 的所有组合 – 点击听取发音
p b j g r m h n c s k  
ぴゃ びゃ じゃ ぎゃ りゃ みゃ ひゃ にゃ ちゃ しゃ きゃ ya
ぴゅ びゅ じゅ ぎゅ りゅ みゅ ひゅ にゅ ちゅ しゅ きゅ yu
ぴょ びょ じょ ぎょ りょ みょ ひょ にょ ちょ しょ きょ yo
注意事项 

  1. 上表的用法与之前相同。将顶部的辅音与右侧的元音相配。例如：きゃ = kya。
  2. 另请注意，因为「じ」读作 / ji /，所以所有小写「や」、「ゆ」、「よ」的发音也都以此为基础，即：/ jya / jyu / jyo /。
  3. 同样的规则也适用于「ち」，它会变成 / cha / chu / cho /；以及「し」，它会变成 / sha / shu / sho /。（不过，也可以认为它们仍是 / sya / syu / syo /。）

小写的「つ」

在两个字符之间插入一个小写的「つ」，可以把第二个字符的辅音带到第一个字符的末尾。例如，如果在「び」和「く」之间插入一个小写的「つ」，组成「びっく」，/ k / 辅音就会被带到第一个字符的末尾，从而读成 “bikku”。同样，「はっぱ」读作 “happa”，「ろっく」读作 “rokku”，以此类推。

示例

 

  1. ざっし (zas-shi) – 杂志
  2. かっぷ (kap-pu) – 杯子
注意事项 

  1. 小写的「つ」用于把第二个字符的辅音带到第一个字符的末尾。例如：「がっき」 = “gakki”。
  2. 增加一个辅音几乎总会产生特有的短促音。不过，一定要用正确的辅音（也就是第二个字符的辅音）来截断发音。

长元音

呼！你快学完了。在最后这一部分，我们将学习长元音，也就是单纯延长一个元音的持续时间。按照下表，根据字符中的元音添加「あ」、「い」或「う」，即可延长该字符的元音。

延长元音
元音 用来延长的字符
/ a / あ
/ i / e / い
/ u / o / う

例如，如果想把「か」变成长元音，就要添加「あ」，组成「かあ」。其他示例包括：「き → きい」、「く → くう」、「け → けい」、「こ → こう」、「さ → さあ」等。这样做的道理很简单。试着分别读出「か」和「あ」，然后尽可能快地把它们连起来读。你很快就会注意到，听起来就像是把 / ka / 拖得比单独读 / ka / 更长。发长元音时，请尽量记住，它们实际上是两个合在一起的音。

一定要确保元音保持得足够长，这一点很重要，因为如果延长得不正确，你说出的可能是“这里”（ここ），而不是“高中”（こうこう）；或者是“中年女士”（おばさん），而不是“祖母”（おばあさん）！

示例

  1. がくせい (ga-ku-se) – 学生
  2. せんせい (sen-se) – 老师
  3. きょう (kyo) – 今天
  4. おはよう (o-ha-yo) – 早上好
  5. おかあさん (o-ka-san) – 母亲

少数情况下存在例外：/ e / 元音会通过添加「え」来延长，或 / o / 元音会通过添加「お」来延长。例如「おねえさん」、「おおい」和「おおきい」。请特别留意这些例外，不过不用担心，它们并不多。
