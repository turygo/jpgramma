将英语单词转换为片假名

只是为了好玩，让我们试着推导一些英语单词的片假名写法。我在下面列出了一些常见规律，但它们只是指导原则，对某些单词可能并不适用。

如你所知，由于日语的音几乎总是由辅音与元音配对组成，任何偏离这一模式的英语单词都会带来问题。下面是一些你可能已经注意到的倾向。

如果你看过电影《迷失东京》，就知道 / l / 和 / r / 是无法区分的。

（１）　Ready -> レディ
（２）　Lady -> レディ

如果连续出现多个元音，或者一个元音音素以 / r / 结尾，它通常会变成长音。

（１）　Target -> ターゲット
（２）　Shoot -> シュート

通常用 / t / 或 / c / 表示的突然截断音要使用小写的「ッ」。

（１）　Catch -> キャッチ
（２）　Cache -> キャッシュ

任何以辅音结尾的单词都需要再加一个元音，以补全辅音—元音模式。（“n”和“m”除外，因为它们可以用「ン」表示。）
对于“t”和“d”，通常加“o”。对于其他所有辅音，通常加“u”。

（１）　Good -> グッド
（２）　Top -> トップ
（３）　Jack -> ジャック
