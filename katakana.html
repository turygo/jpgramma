片假名

官方英文版更新于 2022-02-19 · 查看官方原文

如前所述，片假名主要用于从外语引入的词。它也可以用来强调某些词，作用类似于斜体。如需更完整的用法列表，请参阅维基百科的片假名条目。

片假名表示的语音与平假名相同，只是所有字符都不同。由于外来词必须套入这套有限的［辅音＋元音］音节中，它们会发生许多显著变化，以至于英语母语者有时甚至听不懂那些本应源自英语的词！因此，对英语母语者而言，使用片假名极其困难，因为他们会期待英语单词听起来像……嗯……英语。更好的做法是彻底忘掉原来的英语单词，把这个词当作一个完全独立的日语词来对待；否则，你可能会养成用英语发音说英语单词的习惯（这样一来，日本人可能听得懂，也可能听不懂你在说什么）。

  • japanese-lesson.com

此外，由于日语不使用空格，有时会用符号「・」来表示空格，例如用「ロック・アンド・ロール」表示“摇滚乐”。是否使用这个符号完全是可选的，所以有时什么都不会加。

注意事项

  1. 所有发音都与平假名中的发音完全相同。
  2. 正如我们稍后会学到的，「を」只用作助词，而所有助词都用平假名书写。因此，你几乎永远不需要使用「ヲ」，完全可以忽略它。（除非你在阅读非常古老的电报之类的东西。）
  3. 「シ」、「ン」、「ツ」和「ソ」这四个字符相似得令人抓狂。基本上，前两个比后两个更“横”。它们的小短线斜得更接近水平方向，长线则由下往上画成弧线。后两个的小短线几乎是竖直的，而长线由上往下画，弯曲程度也较小。它几乎像一条斜线，而前者则更像一道弧线。这些字符很难分清，需要一些耐心和练习。
  4. 「ノ」、「メ」和「ヌ」也需要特别留意，「フ」、「ワ」和「ウ」也是如此。没错，它们看起来都非常相似。不，我也无能为力。
  5. 你必须学会正确的笔顺和运笔方向！请使用以下 PDF 练习纸进行练习。

    • japanese-lesson.com
  6. 有时会使用「・」来表示英语中原本会出现空格的位置。

长元音

片假名中的长元音得到了大幅简化。你不必再费劲琢磨元音，只需用这样一条简单的横线表示所有长元音：ー。

示例

  1. ツアー (tsu-a) – 旅行
  2. メール (me-ru) – 电子邮件
  3. ケーキ (ke-ki) – 蛋糕
总结

  • 片假名中的所有长元音都用一条横线表示。例如，“可爱”用片假名写作：「キュート」。

小写的「ア、イ、ウ、エ、オ」

由于平假名音系的限制，多年来人们创造了一些新的组合，用来表示日语中原本没有的发音。最明显的是缺少 / ti / di / 和 / tu / du / 音（因为存在 / chi / tsu / 音），以及除了「ふ」之外没有 / f / 辅音。/ sh / j / ch / 辅音与 / e / 元音的组合也不存在。解决这些缺失音的办法，是加入五个元音的小写形式。/ w / 辅音也采用了这种做法，以取代已经废弃的字符。此外，人们还确立了这样一种惯例：在元音「ウ」上加两小撇（ヴ），再搭配小写的「ア、イ、エ、オ」，表示 / v / 辅音；但这种写法并不常用，可能是因为日本人仍然很难发出 / v / 音。例如，你可能会猜测“volume”应当用 / v / 音来发音，但日本人选择了更容易发音的“bolume”（ボリューム）。同样，vodka 写成“wokka”（ウォッカ），而不是「ヴォッカ」。你可以把“violin”写成「バイオリン」或「ヴァイオリン」。不过其实这并不重要，因为几乎所有日本人无论如何都会把它发成 / b / 音。下表用突出显示的方式列出了新增的缺失音。其他原本就存在的音会酌情复用。

注意事项

  1. 请注意，并没有 / wu / 音。例如，“woman”的片假名写作“u-man”（ウーマン）。
  2. 虽然按照这些规则，/ tu / 音（如“too”中的发音）在技术上可以用「トゥ」表示，但在这些音可用之前就已经普及的外来词，只能用 / tsu / 音凑合。例如，“tool”至今仍写作「ツール」，“tour”同样仍写作「ツアー」。
  3. 很久以前还没有这些新音时，人们别无选择，只能从常规表中直接选用字符，根本不顾实际发音。在老建筑上，你可能仍会看到「ビルヂング」，而不是现代拼法「ビルディング」。

一些片假名词汇示例

把英语单词转换成日语是一门技巧，需要大量练习和一点运气。为了让你了解英语单词是如何“日本化”的，下面列出几个片假名词汇的例子。有时，片假名词汇甚至并不是正确的英语，或者其含义与它本应表示的英语单词不同。当然，并非所有片假名词汇都源自英语。
